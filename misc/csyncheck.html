Simple Syntax Checker for "C" Language

This is just a temporary web page for my simple "C" language. Syntax checker program. Using the CGI (Common Gateway Interface) in the "C" language, to check "C" lanaguage source code. Type some code or copy/paste it into the box below to check it for rudimentary syntax errors. This CGI program will generate a new page with the results on that new page.
